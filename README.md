# 🇫🇷 FrenchUp

אפליקציית תרגול צרפתית ברמת B2/C1 — אתגר יומי עם שינון, תרגום בלחיצה, וקול **נוירוני** (ElevenLabs).
בנויה ב-Next.js, מוכנה לפריסה ב-Vercel.

## מה יש כאן

- **אתגר יומי** ב-4 מיומנויות: דקדוק, אוצר מילים, הבנה, דיבור.
- **בדיקה מקומית** — דקדוק/אוצר מילים נבדקים אוטומטית, הבנה היא רב-ברירה, דיבור מקבל תשובת מודל.
- **תרגום בלחיצה** על כל משפט צרפתי.
- **קול נוירוני** דרך `/api/tts` (המפתח נשאר בצד השרת). אם השרת לא מוגדר — נפילה אוטומטית לקול המכשיר.
- **ניקוד:** תשובה נכונה +50 XP, שגויה −15 XP עם הסבר מלא.

## הרצה מקומית

```bash
npm install
cp .env.example .env.local   # והכנס את מפתח ElevenLabs
npm run dev
```

פתח http://localhost:3000

## פריסה ל-Vercel (אפשר גם מהטלפון)

1. העלה את הפרויקט הזה ל-repo ב-GitHub (ראה למטה).
2. היכנס ל-[vercel.com](https://vercel.com) → **Add New → Project** → בחר את ה-repo.
3. ב-**Environment Variables** הוסף:
   - `ELEVENLABS_API_KEY` = המפתח שלך מ-ElevenLabs (Profile → API Keys).
   - `ELEVENLABS_VOICE_ID` = (לא חובה) מזהה קול מספריית הקולות שלך. ברירת מחדל עובדת בצרפתית.
4. לחץ **Deploy**. תוך דקה האתר באוויר עם הקול הנוירוני פעיל.

> בלי המפתח האתר עדיין יעבוד — פשוט ישתמש בקול המכשיר עד שתוסיף אותו.

## בדיקת תשובות חופשיות (AI)

השאלות הפתוחות (דיבור) נבדקות על ידי Claude דרך `/api/check`: דקדוק, משמעות, האם עונה לשאלה, והאם עומד בתנאים. כדי להפעיל, הוסף ב-Vercel:

- `ANTHROPIC_API_KEY` = מפתח ה-API שלך מ-Anthropic.
- `CHECK_MODEL` = (לא חובה) `claude-sonnet-4-6` (איכות) או `claude-haiku-4-5` (חיסכון).

בלי המפתח, השאלות הפתוחות פשוט יציגו תשובת מודל להשוואה (בלי ציון).
